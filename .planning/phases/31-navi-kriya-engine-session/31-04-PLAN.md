| strings -> DOM | copy comes from the typed `UiStrings` object; no user-supplied text |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-31-09 | Information Disclosure | None — these are presentational components with no data access | accept | NKShape/NKSessionReadout read only typed props; no storage, no network, no PII. React auto-escapes all rendered text. |
| T-31-10 | Tampering | A non-numeric `count`/`target` reaching the readout | accept | Props are typed `number`; TypeScript blocks non-numeric values at the call site. `String(target)` is a safe coercion. No injection sink. |
