## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline label padding |
| sm | 8px | Between link rows in the modal link block |
| md | 16px | Default element spacing, modal section gap |
| lg | 24px | Modal section separator, modal inner padding (p-6 = 24px) |
| xl | 32px | Modal top-level padding on larger viewports (sm:p-7 ≈ 28px, round to 32px) |
| 2xl | 48px | Not used in Phase 6 |
| 3xl | 64px | Not used in Phase 6 |

Exceptions:
- Corner anchor hit area: minimum 44×44px enforced via padding around the visible `Learn` label (Phase 2 D-17 / CONTEXT.md D-04). Padding calculated to fill 44px regardless of text size.
- Touch targets inside modal (links and Close button): minimum 44px height via `min-h-[44px]`.

### New: LearnDialog (`src/components/LearnDialog.tsx`)

| Property | Specification |
|----------|---------------|
| Element | Native `<dialog>` with `.modal-fade` class |
| Shape | `max-w-sm` (same as EndSessionDialog / ResetStatsDialog) |
| Border radius | `rounded-3xl` |
| Border | `border border-teal-100` |
| Background | `bg-white` |
| Shadow | `shadow-[var(--shadow-breathing-card)]` |

| Open behavior | `dialog.showModal()` — top-layer, focus trap, Esc cancels |
| Esc behavior | Fires `cancel` event → `onClose()` callback |
| Backdrop click | Click on `<dialog>` element itself (not inner panel) → `onClose()` |
| Default focus | `Close` button (NOT a Forrest link — D-05) |

| Reduced motion | `.modal-fade` already gated by `@media (prefers-reduced-motion: reduce) { transition: none !important }` in `theme.css` |

#### LearnDialog internal layout (scroll order = DOM order)

1. `<h2>` — modal title (e.g. `About this practice`)
2. Explainer section (D-08) — three subsections in fixed order:
   - "What is HRV / resonance breathing" (2–3 sentences, `text-base text-slate-700`)
   - "How this app times your breath" (2–3 sentences, `text-base text-slate-700`)
   - "Who is Forrest Knutson" (1–2 sentences, `text-base text-slate-700`; MUST contain verbatim phrase `inspired by Forrest's teachings` — D-11)
3. Link block (D-12) — five items, plain labeled text links, `text-sm` weight regular; each `<a target="_blank" rel="noopener noreferrer">`:
   - YouTube channel
   - Website
   - Book (`https://amzn.to/3RTAVqi` — D-12 locked URL)
   - Hero video (URL supplied at plan-phase)
   - Key videos (2–5 items, URLs supplied at plan-phase)
4. Medical-advice sentence (D-14 #1): `This is guided breathing practice — not medical advice.` — `text-sm text-[var(--color-breathing-muted)]`
5. Affiliation micro-line (D-14 #2): `Independent project. Not affiliated with Forrest Knutson.` — `text-xs text-[var(--color-breathing-muted)]`
6. Close button — `min-h-[44px]`, `rounded-full`, `border border-teal-200 bg-white text-teal-800 font-semibold`, hover teal-50, active teal-100; focus-visible ring accent

## Interaction States

### Corner Anchor

| State | Visual |
|-------|--------|

|-------|----------|

| Close button click | `onClose()` |
| External link click | Opens new tab; does NOT close modal; does NOT navigate practice tab |
| Reduced motion | `transition: none` on `.modal-fade` (existing theme.css rule) |

## Accessibility Contract

| Requirement | Implementation |
|-------------|---------------|
| WCAG 2.1 AA color contrast | Accent `#0f766e` on white passes 4.5:1 at text-sm. Muted `#64748b` on white passes 4.5:1 at text-xs (verify during UAT). |
| Keyboard operability | Corner anchor: focusable button. Modal: `<dialog>` native focus trap; Esc cancels; Close button receives default focus. All links keyboard-reachable within the modal. |
| Screen-reader | `<dialog>` has `aria-labelledby` pointing to the modal `<h2>`. Corner anchor has `aria-disabled="true"` during session. External links carry visible text labels (no "click here"). |
| Focus visibility | All interactive elements use `focus-visible:ring-2 focus-visible:ring-breathing-accent focus-visible:ring-offset-2` (Phase 2 D-21) |
| Touch targets | Corner anchor: min 44×44px (D-04). Close button: `min-h-[44px]`. Link rows: `min-h-[44px]` padding. |
| External link safety | `target="_blank" rel="noopener noreferrer"` on every external link (D-07) |
| Reduced motion | Dialog fade disabled by existing `@media (prefers-reduced-motion: reduce)` rule in `theme.css`. No new animation introduced. |

## Layout Contract

### Corner Anchor Placement

```
<main>                          ← existing page root
  [corner anchor]               ← new; position: absolute/fixed top-4 right-4
  <section>                     ← existing breathing card + session flow
    …
  </section>
  <StatsFooter>                 ← existing; unchanged
  <EndSessionDialog>            ← existing
  <ResetStatsDialog>            ← existing
  <LearnDialog>                 ← new; rendered last inside <main>
</main>
```

- Anchor must not overlap the breathing card at any breakpoint.
- On narrow viewports (< 375px), anchor shrinks to `text-xs` if needed, but hit area remains 44×44px.
- Safe-area insets: anchor uses `top-[max(16px,env(safe-area-inset-top))] right-[max(16px,env(safe-area-inset-right))]` or equivalent to respect iOS notch/dynamic island.
