## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| picker `onChange` → choice-hook setter | The setter is the only path that writes to `localStorage` + dispatches `hrv:prefs-changed`. The picker holds no input validation responsibility — TypeScript's typed union (`BreathingShapeVariant` / `RingCueStyle`) prevents invalid values at the compiler level. |
| picker JSX → React render output | `strings.*` values are interpolated into button labels via JSX; React 19 auto-escapes. The `strings` prop is supplied by the parent (Plan 04 supplies from `UI_STRINGS`). No user input ever flows here. |
