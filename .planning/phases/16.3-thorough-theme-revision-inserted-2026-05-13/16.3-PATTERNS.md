```

2. Open the app in Chrome (preferred — same engine used for 16.1/16.2 UAT).
3. Open SettingsDialog via the gear icon.
4. For the palette under test, select it in the ThemePicker, close the dialog, and:
   a. Observe the orb at idle. Verify the In gradient feels integrated with the surrounding chrome (bg, surface, accent).
```
