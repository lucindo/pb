### Computed Contrast (Pre-Edit Calculator)

Inline `node -e` calc mirroring `theme.contrast.test.ts:20–90` math (sRGB→linear with 0.03928 threshold; 0.2126/0.7152/0.0722 channel weights; `(max+0.05)/(min+0.05)` ratio).

| Invariant | Result | Floor | Verdict |
|-----------|--------|-------|---------|
| **Polarity** (inMidLum > outMidLum) | inMidLum = 0.463, outMidLum = 0.078 | inMidLum > outMidLum | **✓ pass** (In is the light pole, ratio of luminances ≈ 5.9×) |
| **THEME-05** (orb-in mid vs orb-out mid) | 4.02 | ≥ 1.5 | **✓ pass** (well above floor; matches indicative RESEARCH A2 figure ~4.5 to within rubric-choice tolerance) |
| **D-01** (accent-strong vs on-accent) | 5.60 (mauve `#ca9ee6` vs base `#303446`) | ≥ 1.5 | **✓ pass** (well above floor) |
| orb-in-text legibility (crust `#232634` vs in-mid) | 9.02 | ≥ 3.0 large-display, ≥ 4.5 body | **✓ pass** AA body |
| orb-out-text legibility (text `#c6d0f5` vs out-mid) | 9.43 | ≥ 3.0 large-display, ≥ 4.5 body | **✓ pass** AA body |

Inline calc command (reproducible):
```bash
node -e '
function s2l(c){c=c/255;return c<=0.03928?c/12.92:Math.pow((c+0.055)/1.055,2.4);}
function lum([r,g,b]){return 0.2126*s2l(r)+0.7152*s2l(g)+0.0722*s2l(b);}
function hex(h){h=h.replace("#","");return [parseInt(h.slice(0,2),16),parseInt(h.slice(2,4),16),parseInt(h.slice(4,6),16)];}
function mid(a,b){return [Math.round((a[0]+b[0])/2),Math.round((a[1]+b[1])/2),Math.round((a[2]+b[2])/2)];}
function cr(a,b){const la=lum(a),lb=lum(b);const x=Math.max(la,lb),y=Math.min(la,lb);return (x+0.05)/(y+0.05);}
const im=mid(hex("#babbf1"),hex("#8caaee")), om=mid(hex("#51576d"),hex("#414559"));
console.log("THEME-05=",cr(im,om).toFixed(2),"D-01=",cr(hex("#ca9ee6"),hex("#303446")).toFixed(2),"polarity(in>out)=",lum(im)>lum(om));'
```

### Rubric Ambiguity Flagged (for RESEARCH amendment in Plan 07 if desired)

The RESEARCH §Pattern 2 row for `bg-soft` reads: _"mantle / surface0 (one tier deeper than bg for Light, one tier lighter than bg for Dark)"_. In Frappé, `mantle` is **darker** than `base` (luminance 0.022 vs 0.038) — so "mantle" is the deeper option, not the lighter one. For Dark palettes the Pattern 2 example listed in this Plan 01 task description (`bg-soft → mantle`) contradicts the rubric description ("one tier lighter than bg for Dark"). This dry-run resolves the ambiguity by selecting `bg-soft = surface0` (one tier lighter than bg, matching the current Dark theme's bg<bg-soft direction). **Recommendation for Plans 03–06:** prefer the rubric description ("lifted from bg in Dark") over any literal source-role-name suggestion when they conflict.
