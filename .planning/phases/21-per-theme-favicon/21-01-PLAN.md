## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| build-time source → runtime SVG data-URI | The 5 favicon hex values are hardcoded constants authored at build time. No user-supplied or network input crosses into the favicon construction. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-21-01 | Tampering / Injection | `buildFaviconDataUri` SVG string interpolation | mitigate | The interpolated fill values are a closed set of 5 hardcoded 6-digit hex literals from `FAVICON_COLORS`, never user input. `buildFaviconDataUri` accepts only `Exclude<ThemeId,'system'>` (a 5-value union), so no arbitrary string can reach the template. No SVG-injection surface exists. |
| T-21-02 | Information disclosure | favicon data-URI contents | accept | The data-URI embeds only a flat-circle SVG and a public theme color. No PII, secrets, or sensitive data. Low-value target. |
| T-21-03 | Tampering | hand-copied hex map drift (faviconPalette ↔ theme.css ↔ index.html) | mitigate | `favicon.sync.test.ts` (Task 2) fails the build if any of the three hex copies diverge — drift is caught by the per-commit green-gate, not shipped. |
</threat_model>
