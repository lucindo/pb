## Deviations from Plan

### 1. [Rule 3 — Blocking issue] Vitest API ordering: `describe.skip.each`, not `describe.each.skip`
- **Found during:** Task 3 first test run
- **Issue:** The plan's `<acceptance_criteria>` and `<verification>` both reference `describe.each.skip(...)`, but Vitest 4.1.5 throws `TypeError: describe.each.skip is not a function`. The correct ordering is `describe.skip.each(...)` (skip modifier applied first, `.each` second).
- **Fix:** Updated the test file to `describe.skip.each(BANNED_PATTERNS)(...)` and added an inline comment documenting that the API order matters in Vitest 4. The two forms are functionally identical (suite-level skip).
- **Files modified:** `src/styles/theme.no-hardcoded-classes.test.ts`
- **Commit:** 77bbc19 (same commit as Task 3)
- **Impact:** None — the acceptance criterion "describe.skip" pattern grep matches the new form.

### 2. [Rule 1 — State integrity] `requirements.mark-complete` reverted post-application
- **Found during:** Post-execution state updates
- **Issue:** The execute-plan workflow's `<state_updates>` block says "Mark completed requirements from PLAN.md frontmatter". The plan's frontmatter lists `requirements: [THEME-UI-01]`, but THEME-UI-01 is a phase-spanning requirement (the full migration goal) that completes at plan 07's phase-close gate, NOT at the end of this preflight plan. The SDK handler correctly flipped both `[ ] -> [x]` and `Pending -> Complete` based on the frontmatter, but this would falsely advertise the requirement as done across the v1.1 milestone.
- **Fix:** Reverted both edits — the requirement is back to `[ ]` and traceability `Pending`. Plan 07 will re-flip when the guard test goes GREEN.
- **Files modified:** `.planning/REQUIREMENTS.md` (two single-character reverts on the THEME-UI-01 lines)
- **No commit yet:** captured in the final metadata commit at plan close.

No other deviations. Token seed values were not retuned (D-01), AP-07 sentinel held (zero existing-line edits to `theme.css`), and the per-commit green-gate is intact (D-17).
