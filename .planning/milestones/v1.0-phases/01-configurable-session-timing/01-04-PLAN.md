## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Running session state → visible guidance | Derived session state determines what users follow during breathing practice. |
| End confirmation → lifecycle transition | User confirmation controls whether a timed running session is stopped. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-10 | T | `BreathingShape` | mitigate | Consume `SessionFrame` only; do not create an independent CSS/JS timer that could drift from text/time state. |
| T-01-11 | D | `SessionReadout` | mitigate | Clamp completed timed display to zero remaining and rely on hook cleanup to stop RAF after completion. |
| T-01-12 | R | Timed end confirmation | mitigate | Require explicit confirmation for timed sessions; cancellation leaves state unchanged and is covered by tests. |
</threat_model>
